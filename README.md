## 🕶 Что это такое

**Telegram Channel Parser** — это асинхронный инструмент  
для **поиска, проверки и анализа Telegram-каналов и групп**  
с полноценной **веб-панелью и live-логами**.

Проект создавался как **рабочий инструмент**,  
а не демонстрационный скрипт.

Без ботов.  
Без скрытых автозапусков.  
Без «магии».

Только контроль и прозрачная логика.

---

## 🎯 Назначение

- Анализ Telegram-нишей
- Поиск каналов и групп по ключевым словам
- Сбор информации о подписчиках
- Определение типа: `channel / group`
- Подготовка данных для ресёрча и аналитики

---

## 🧠 Возможности

- 🔐 **Telegram-авторизация**
  - API ID / API HASH
  - Код из Telegram
  - Поддержка 2FA
  - Сессия сохраняется и не сбрасывается

- 🔍 **Поиск через Telegram API**
  - Несколько ключевых слов
  - Кириллица и латиница
  - Удаление дубликатов

- 🧪 **Проверка каналов**
  - Название
  - Количество подписчиков
  - Тип: channel / group
  - Retry при пустом HTML
  - Рандомизация User-Agent
  - Контролируемые задержки

- 🔁 **Перепроверка пустых**
  - Только проблемные записи
  - Без перезапуска поиска
  - Без потери данных

- 🧵 **Live-панель**
  - WebSocket
  - Прогресс выполнения
  - Логи в реальном времени
  - Без перезагрузки страницы

- 🗑 **Очистка результатов**
  - Чистит только базу результатов
  - Не влияет на Telegram-сессию
  - Не запускает pipeline повторно

- 📤 **Экспорт в Excel**
  - `.xlsx`
  - Сортировка по подписчикам
  - Готово к дальнейшей аналитике

## ⚙ Как это работает

1. Пользователь вручную запускает поиск
2. Telegram API возвращает чаты по запросам
3. Username собираются и дедуплицируются
4. Каждый username проверяется через `t.me`
5. Из HTML извлекаются:
   - название
   - подписчики
   - тип
6. Результаты сохраняются в SQLite
7. UI обновляется через WebSocket

❗ Автозапуски исключены на уровне архитектуры
